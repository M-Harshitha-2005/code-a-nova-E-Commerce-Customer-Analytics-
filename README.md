## Abstract
RetailMind Analytics is an end-to-end Business Intelligence and Customer Analytics platform developed to help organizations understand customer purchasing behaviour and convert raw retail transaction data into actionable business insights.
The project integrates Data Analytics, Machine Learning, Business Intelligence, Customer Segmentation, Churn Analysis, Product Recommendation, and Interactive Storytelling into a unified Streamlit application. Rather than presenting static reports, the platform enables users to explore business performance dynamically through interactive dashboards, intelligent visualizations, and data-driven recommendations.
The system demonstrates a complete analytics workflow beginning with data preprocessing and exploratory analysis, followed by customer segmentation using RFM Analysis, churn identification, collaborative filtering-based product recommendation, and executive-level business storytelling. These components collectively support strategic decision-making by helping businesses improve customer retention, optimize marketing campaigns, identify high-value customers, and enhance overall operational performance.
RetailMind Analytics showcases the practical application of Data Science and Machine Learning techniques in solving real-world retail business problems while providing an intuitive and professional user experience through an interactive web-based dashboard.
## Project Overview
RetailMind Analytics is a comprehensive Customer Intelligence platform built for the E-Commerce domain. The application analyzes customer transaction data to uncover hidden business patterns, monitor key performance indicators, evaluate customer behaviour, identify potential churn, recommend products based on purchasing similarity, and present strategic business insights through interactive dashboards.
The platform combines multiple analytical techniques—including Exploratory Data Analysis (EDA), RFM-based Customer Segmentation, Collaborative Filtering, Churn Analysis, and Business Storytelling—to provide a complete decision-support system for business users.
Designed using Python and Streamlit, the project emphasizes interactive visualizations, real-time filtering, intuitive dashboards, and actionable insights, making complex retail data easier to interpret for both technical and non-technical users.
## Business Problem

In the E-Commerce industry, businesses collect a large amount of customer transaction data every day. However, this data is often difficult to understand and use effectively without proper analysis.
Many businesses face challenges such as:
- Understanding customer purchasing behaviour.
- Identifying valuable customers.
- Finding customers who may stop purchasing.
- Recommending suitable products to customers.
- Tracking business performance through meaningful insights.
Without proper analytics, it becomes difficult to make informed business decisions, improve customer satisfaction, and increase sales.
## Proposed Solution
RetailMind Analytics was developed to solve these challenges by converting raw customer transaction data into meaningful business insights.
The application combines Data Analytics, Machine Learning, and Interactive Dashboards to help businesses:
- Analyze customer purchasing behaviour.
- Segment customers based on their buying patterns.
- Identify customers with a high risk of churn.
- Recommend products using customer purchase history.
- Monitor important business metrics through interactive dashboards.
- Support better business decisions with clear visualizations and reports.
The platform is designed with a simple and user-friendly interface so that both technical and non-technical users can easily explore business data.

## Installation Guide

Follow the steps below to run the project on your local computer.

## Step 1: Clone the Repository
```b
git clone https://github.com/M-Harshitha-2005/code-a-nova-E-Commerce-Customer-Analytics-.git
```
### Step 2: Open the Project Folder
```
cd code-a-nova-E-Commerce-Customer-Analytics-
```
### Step 3: Create a Virtual Environment (Optional)
Windows:
```
python -m venv venv
```
Activate the virtual environment:
```
venv\Scripts\activate
```
### Step 4: Install Required Libraries
```
pip install -r requirements.txt
```
### Step 5: Run the Streamlit Application
```
streamlit run Home.py
```
The application will automatically open in your default web browser.
